## 接口依赖关系

```
接口1（GET /api/xxx）→ 响应字段 data.id → 接口2（POST /api/yyy/{id}）的路径参数 id
接口2（POST /api/yyy/{id}）→ 响应字段 data.flowId → 接口3（GET /api/flow/{flowId}）的路径参数 flowId
```

> 孤立接口（入口接口，直接接受用户输入）：接口1
