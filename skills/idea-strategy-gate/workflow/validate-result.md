# 阶段六：结论校验与独立复核（扩展版）

## 输入信息

**必须入参**：
- `结论报告`：阶段五产物。
- `物种识别记录`：阶段一产物。
- `概念清晰度评估`：阶段一产物。
- `用户原始需求`：用于核对是否偏离。
- `全局约束`：执行前判断，不进入 MVP 或实现。

**上下文参考**：
- [Idea 物种识别参考](../references/idea-species.md)
- [概念清晰度评估参考](../references/concept-clarity.md)
- [语气校准参考](../references/tone-calibration.md)

## 目标要求

**任务**：检查结论是否真的完成方向校验，而不是披着战略语言的执行建议；同时检查物种识别正确性、概念清晰度处理正确性、语气中立性。

**完成标准**：
- 校验清单逐项有通过/不通过。
- 任一关键项不通过时，回退对应阶段修正。
- 高风险情形已触发独立 subagent 复核。

## 主控校验清单

### 第一组：结论结构校验（保留原有）

```markdown
## 结论结构校验

- [ ] 结论明确：多 idea 有主推/观察/暂缓或否决；单 idea 有通过/暂存/否决/前置概念重构/前置研究规划。
- [ ] 没有进入执行层：未提出 MVP、功能列表、技术实现、冷启动、增长打法、时间表（注意：方法论物种的「手动跑通」不算执行层）。
- [ ] 用户目标函数明确：长期主线、价值排序或不可牺牲项被写入判断依据。
- [ ] 每个主推/通过项都有战略理由，不只是「喜欢、热门、容易做、短期可变现」。
- [ ] 每个暂缓/否决项都有具体否决逻辑或缺失事实。
- [ ] 已识别机会成本：推进该方向会挤掉什么。
- [ ] 已识别最大反方风险：至少一条根本前提错误会导致方向不成立。
- [ ] 若信息不足，结论被降级为观察/暂存，并列出必须澄清的问题。
- [ ] 若 Top 候选重叠，已说明并列、主从、合并或冲突关系。
- [ ] 若存在路径依赖，已区分当前优先级与长期战略价值。
```

### 第二组：物种识别与适配校验（新增）

```markdown
## 物种识别校验

- [ ] 阶段一是否对每个候选完成了五维度物种判断？
- [ ] 多物种 Idea 是否被显式拆分为核心 + 外壳？
- [ ] 阶段三/四是否使用了对应物种的问题集？
- [ ] 是否避免了用一种物种的标准（如 PMF）评估另一种物种（如方法论）？
- [ ] 阶段五结论是否显式说明了物种？
- [ ] 「推荐下一步路径」是否与物种匹配？
  - 方法论物种：是否推荐了「手动跑通」而非「MVP」？
  - 研究物种：是否推荐了「研究路径设计」而非「商业模式」？
  - 基础设施物种：是否基于「下游真实调用次数」判断？
```

### 第三组：概念清晰度处理校验（新增）

```markdown
## 概念清晰度校验

- [ ] 阶段一是否列出了每个候选的核心概念并评级？
- [ ] L0/L1 概念的 Idea 是否触发了正确的路由？
  - 产品物种 + L0/L1 → 是否触发「前置概念重构」？
  - 研究物种 + L0/L1 → 是否调整问题集（追问研究路径、证伪标准）？
  - 方法论物种 + L1 → 是否用「手动跑通」作为升级路径？
- [ ] 是否避免了把「需要研究」字面理解为「核心资产缺失」？
- [ ] 是否避免了把「概念尚未形式化」误判为「方向不可行」？
```

### 第四组：语气校准校验（新增）

```markdown
## 语气中立校验

- [ ] 输出中**不**包含动机攻击词：虚荣心、心动陷阱、宏大叙事陷阱、自我感动、逃避执行、新鲜感作祟、热点跟风、绑架。
- [ ] 输出中**不**包含命令式表达：你应该 / 你必须 / 你不能 / 你只能 / 你必须放弃。
- [ ] 输出中**不**包含道德评判：你不应该这么想 / 这种想法很危险 / 你犯了错误。
- [ ] 输出中**不**包含隐喻夸大：深坑 / 死路 / 空中楼阁 / 浪费时间。
- [ ] 每个判断都附有可观察的事实或可追问的问题。
- [ ] 每个否决/暂缓结论都附有「重评条件」或「最小验证动作」。
- [ ] 若用户表现强偏好，结论是否在尊重偏好的前提下保留缺口提示？
```

### 第五组：常见回答处理校验（新增）

```markdown
## 二次追问校验

- [ ] 若用户回答过「需要研究」，是否执行了二次追问（研究路径 / 证伪标准 / 第一里程碑）？
- [ ] 若用户回答过「这是执行问题」，是否执行了二次追问（前提是否独立 / 是否已验证）？
- [ ] 若用户回答过「需要能够 X」，是否执行了二次追问（路径 / 证据 / 失败时方向是否仍成立）？
- [ ] 二次追问的判定结果是否被纳入最终结论？
```

## 独立复核触发条件

符合任一项时，必须委派 subagent：
- 结论影响用户数月以上投入。
- 用户对被否决 idea 明显防御。
- 候选方向涉及创业、职业主线、内容主轴、基础设施。
- 主控对 Top 排序置信度不足。
- **新增**：物种识别存在不确定性（混合物种、用户表达模糊）。
- **新增**：核心概念在 L0/L1 但未触发前置重构（需复核是否合理）。

## 回退规则

- 未明确目标函数：回退阶段一。
- 未完成物种识别或物种识别错误：回退阶段一。
- 未评估概念清晰度或评估错误：回退阶段一。
- 候选定义混乱：回退阶段二。
- 缺战略理由 / 用错物种问题集：回退阶段三。
- 未压测 / 未对模糊回答二次追问：回退阶段四。
- 结论结构不完整 / 推荐路径不匹配物种 / 措辞违反语气校准：回退阶段五。
