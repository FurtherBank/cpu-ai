## 11. 结构校验清单

- [ ] 原始信息已保全或说明无法保全的原因。
- [ ] 净化正文中的强事实和强规则能回溯来源。
- [ ] 事实、推断、假设、待确认、历史背景、候选方案没有混写。
- [ ] 核心概念具备定义、边界、非范围、生产方、消费方、下一步判断。
- [ ] 权威来源和责任边界明确。
- [ ] 关系已经显式化，不只是一组并列段落。
- [ ] 领域特定检查项已应用或已生成临时检查项。
- [ ] Skill 已处理的通用问题和使用者需要继续判断的 case 专项问题已分开记录。
- [ ] 需要升级才能优化的问题已进入升级候选清单。
- [ ] 目标消费路径下，读者能直接使用结果。
- [ ] 剩余风险和补充信息请求已记录。
