# 净化信息包模板

本模板分为两类内容：

- **产物本体**：面向目标读者的最终正文，可被单独抽取、发布、复制或替换原结构。
- **净化日志记录**：记录本 Skill 的执行依据、结构改造、追溯校验、风险交接和主控自检，不属于最终正文。

### 12. 主控结构校验记录

- [ ] 产物本体可单独抽取，且不依赖净化日志才能被目标读者消费。
- [ ] 产物本体是可直接消费的新结构，不只是信息清单或分析报告。
- [ ] 产物本体对原结构做了实质性重排、合并、拆分、上提、下沉或隔离。
- [ ] 已形成原结构诊断和目标结构判断，并记录在净化日志中。
- [ ] 已提供旧结构到新结构的映射，并说明每类关键原始信息的去向。
- [ ] 原始信息已保全或说明无法保全的原因。
- [ ] 产物本体中的强事实和强规则能回溯来源。
- [ ] 事实、推断、假设、待确认、历史背景、候选方案没有混写。
- [ ] 核心概念具备定义、边界、非范围、生产方、消费方、下一步判断。
- [ ] 权威来源和责任边界明确。
- [ ] 关系已经显式化，不只是一组并列段落。
- [ ] 领域特定检查项已应用或已生成临时检查项。
- [ ] Skill 已处理的通用问题和使用者需要继续判断的 case 专项问题已分开记录。
- [ ] 需要升级才能优化的问题已进入升级候选记录。
- [ ] 剩余风险和补充信息请求已记录。
