# 外部验收断言清单

供第三方（用户或另一个 agent）只读三本账与研究产出执行。每条断言形如：若协议真实生效，工件上应出现 X；X 缺失或呈 Y 形态，说明对应条款未生效或正被吞噬。

前置条件：验收者可读 git 全历史与最终研究产出。标注【实证】的断言可机械判定；标注【推断】的依赖分布形态判读，需裁量。

## 工件层

- **只追加不改写**【实证】：git 历史为纯追加链，任一旧条目内容在后续 commit 中逐字节不变；出现改史（rebase/force-push、hash 链断裂、与审计记录中 HEAD hash 不符）即失效。
- **账目存续**【实证】：活跃工作期账面有增量；空账期审计输出「不可判定」，连续两次后升级为告警。长期空账无升级 = 协议已静默死亡。

## 预注册层

- **升档即预注册**【实证】：工作档/地基档条目的 commit 时间戳早于该判断被任何后续条目或产出引用的首次时间；时间戳成批集中或引用先于登记 = 批量补写。
- **分型分布**【推断】：长期研究账面呈档位混合分布，工作档与地基档随复用事件非零增长；全账低档而补丁与产出持续增长 = 永不升档路由。
- **失效条件可证伪**【推断】:抽任一在册失效条件，能读出至少一个具体可观察触发事件；普遍为「仍有用/未被推翻」类措辞 = 空洞失效条件。
- **预期差分承重**【推断】：差分与对手判断/默认预期可分辨，且存在被后续现实事件确认或否证的引用记录；差分零兑现率 = 伪预测充数。

## 补丁层

- **补丁识别**【实证】：在册判断的语义变体条目均反向引用原条目 ID 且有对应补丁台账条目；diff 出无补丁的近似重复条目 = 暗补丁（「澄清表述」式修改）。
- **异常先核失效条件**【实证】：每条补丁的消解链首环是失效条件核查记录；补丁存在而消解链缺失时，审计记录必见「视同命中」。
- **独立新预测真实性**【推断】：该字段的预测有兑现或否证记录；呈「N 条无 + 1 条琐碎预测」周期图样 = 连击重置。
- **禁止力蠕变**【实证】：修改失效条件的补丁自报「扩/缩/不变」与文本 diff 一致；不符即伪证。

## 审计层

- **周期审计**【实证】：审计记录数 ≥ ⌊预注册总数/N⌋，每条含触发原因。
- **审计可重算**【实证】：每条审计记录含原始计数值，第三方按账面重算一致；缺数值或不符 = 伪审计。
- **孕育期门控**【实证】：观察标记出现后下次审计间隔减半（时间戳可验）；连续两次观察＋锚点零增后必有告警条目；观察标记连续 ≥3 而无告警 = 话术续期。
- **锚点真实性**【推断】：斜率门控计入的外部锚点均被至少一条在册判断的依赖前提引用；大量无引用锚点 = 装饰性锚点通胀。

## 退出层

- **告警冻结**【实证】：告警 commit 与 exit 闭合 commit 之间只有退出类字段追加。
- **回滚点可审**【实证】：exit 条目载明回滚点 ID、判据、降级清单与数量；历次退出降级数恒零而账龄长 = 回滚点操纵（协议要求此时强制回滚初始承诺集）。
- **待验禁引用**【实证】：exit 之后新条目的依赖前提中无待验状态 ID。
- **现实绑定有限期**【实证】：前 k 个待验各有含有限截止的绑定事件；截止普遍远期或缺失 = 无限展期；逾期应见「隔离」条目。

## 结算层（结算形态适用）

- **结算覆盖连续性**【实证】：结算条目的产出 commit 区间首尾衔接无缺口；缺口或长期无结算条目 = 结算停摆（协议要求按「不可判定」处理，连续两次升级告警）。
- **弹回闭环**【实证】：每条弹回点在下一结算条目前有对应处理痕迹（补失效条件、降级、补丁或废弃）；弹回长期挂起 = 结算被架空。
- **检出完备性**【推断】：抽取任一产出段，由第三方复检锚点事件（新判断/复用/无来源前提/变体/未申报异常/归因/健康总评），与该段结算条目的检出清单比对；系统性漏检 = 结算方失能或与研究方同构漂移。
- **异常检出对照**【推断】：产出中与在册预期差分相抵触的落盘观察，应在对应结算条目的弹回清单出现「未申报异常」项并有后续补丁；抵触观察存在而弹回清单持续为空 = 结算方对协议最重要的检出义务失能。

## 边界层

- **直觉豁免非幸存者偏差**【实证】：直觉登记时间戳先于验证事件，命中率记录含未命中条目；全命中且事后登记 = 只登记赢的。
- **承诺型有界**【推断】：承诺型条目占比有界且每条含可观察退出条件；占比激增或退出条件空洞 = 豁免通道滥用。

## 已知不可判定区

- 评估感是否真实发生（内部状态，周期审计兜底）。
- 账外复用在产出不带条目 ID 时不可见（引用规则是唯一防线）。
- 「独立新预测」的差分性真伪最终依赖写入端；本清单只能捕获其兑现率异常。
