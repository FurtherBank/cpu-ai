# 注意力预算接入

本 Skill 直接受 [`f-attention-budget-constraint`](../../f-attention-budget-constraint/SKILL.md) 约束，不复制其完整协议。

## 常见承重内容单元

- 整体 OK、责任边界、不可接受损失和风险政策。
- 任务形态选择、事实冲突和权利判断。
- 每个独立候选生成源。
- 高严重度义务的定义、证据与失效条件。
- 关系、不变量、权衡和整体闭包。
- 证据有效性、独立性和下游关口。
- 每类充分性逃逸失败。

## 运行要求

- 不同确认方式会改变后续走向时，先局部聚焦。
- 单点额外推导达到千字级、上下文受污染或高风险不可后置校验时，一承重内容单元一 Subagent。
- Subagent 只收到该单点必要上下文，不读取竞争分支结论。
- 主控先检查单点返回的结论、证据、备选、边界、未知和摘要，再做关系判断。
- 在全部必要单点封口前，不得输出综合质量范围。
- 派发 Prompt、返回路径、fork 情况和整合前关系判断进入行为日志。
