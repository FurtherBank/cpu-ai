
# 字幕转换
将字幕内容完整保留并转化为完整中文文章，然后写回到原文件。


注意，最后形成的段落开头，也注明下内容对应时刻，方便 refer 回视频查找。
可参考以下描述格式：
```
[09:28-09:52] 这部分内容xxxx
```

另外，如果你认为视频或对话中的某个时间点，用户一定需要配合视频附图进行理解，你可以在对应位置做个标记：
```
[🖼️ 附图描述 09:40]
```

## 注意事项类口播：结构化表达

提醒、限制、禁忌、**常见错误**、纠错口令等，译成中文时宜用**加粗小节名 + 列表**，或 **现象 / 对策**、**不要 / 宜** 等对偶块呈现，便于扫读与执行；**不得**因排版而删掉字幕里的条件、数字、否定句，也**禁止**编造字幕未出现的对策。完整规则见：`.cursor/skills/youtube-srt-transform/references/srt-transform-rules.md`（与本 Skill 内 `srt-transform-rules.md` 一致）。
