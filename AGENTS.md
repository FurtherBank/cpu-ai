# 仓库说明

- 这个仓库对应 `.cursor` 目录下所有通用的 AI 套件
- 预期这个仓库最终 clone 到工作区根目录的 `.cursor` 路径使用

## Skills 目录

- 本仓库的规范技能存放在 `skills/`；通过 `npx skills add` 安装的副本在 `.agents/`（已在 `.gitignore` 中忽略）

### 软链接引用的技能

若需要通过**软链接（符号链接）**在 `skills/` 下引用本仓库外的技能目录（例如指向其他项目或本机路径），须同时满足：

1. 在 `.gitignore` 中登记该链接路径（相对于仓库根目录），例如 `skills/my-external-skill/`
2. 不在仓库内提交链接目标的真实文件内容；仓库只保留链接本身或由协作者按文档自行创建链接

新增或变更软链接技能时，**必须先更新 `.gitignore` 再提交**，避免将外部路径下的文件误纳入版本库。
